Если перестал обновляться файл resolv.conf на mint

Можно попробовать так:

sudo dpkg-reconfigure resolvconf

Или так:

sudo mv /etc/resolv.conf /run/resolvconf/resolv.conf
sudo ln -s /run/resolvconf/resolv.conf /etc/resolv.conf

Мне помог второй способ.

Еще вариант:

В /etc/NetworkManager/NetworkManager.conf закомментируйте dns=dnsmasq
sudo dpkg-reconfigure resolvconf 
sudo resolvconf --enable-updates
Fri, Mar 13, 2015
TypeError: $(...).tipuesearch is not a function

Решение:

Устанавливаем поиск по инструкции с сайта http://www.tipue.com/search/docs/. Но! Функция показа результатов должна быть такой:

<script>
var $j = jQuery.noConflict();
$j(document).ready(function() {
     $j('#tipue_search_input').tipuesearch({
         'show': 10,
         'mode': 'json',
         'contentLocation': '{{ SITEURL }}/tipuesearch_content.json'
     });
});
</script>
