About

Keybase.io

Keys for everyone!

Keybase maps your identity to your public keys, and vice versa.

Cheat Sheet

Common cmds

keybase login         # or signup
keybase prove twitter # prove thyself
keybase prove github  # etc.
keybase encrypt ewok -m "a secret"
keybase sign -m "hello world"

# The new Keybase filesystem
# https://keybase.io/docs/kbfs

cd /keybase/private/ewok
echo "a secret for me" > diary.txt

cd /keybase/public/ewok
echo "I abdicate the throne" > signed_by_me.txt

Follow smb

(ex: hashicorp)

keybase id hashicorp
keybase follow hashicorp
keybase encrypt hashicorp -m "a secret msg"

# The new Keybase filesystem
# https://keybase.io/docs/kbfs

cd /keybase/private/ewok,hashicorp
echo "Pst. My CC is 4147..." > ewok_cc.txt
cp ~/FAilVid-ewok-and-hashicorp.mp4 .

# any public, signed files?
ls /keybase/public/hashicorp

Working with data

# view someone else's signed files
cd /keybase/public/chris
cat plan.txt

# your encrypted, private directory
cd /keybase/private/ewok
echo "I have a secret" > diary.txt

# your public, signed directory
cd /keybase/public/ewok
echo "This is signed." > a_contract_with_the_world.txt

# an encrypted end-to-end directory with a friend
cd /keybase/private/ewok,chris
echo "Wicked." > awes.txt
