Любимые горячие клавишы в pycharm

  • New File - Alt+Ins
  • Settings - Ctrl+Alt+S
  • Find Usage - Alt+F7
  • Select Word - Ctrl+W
  • Quick Definition - Ctrl+Shift+I
  • Quick Documentation - Ctrl+Q
  • Parameters - Ctrl+P

Find

  • Class - Ctrl+N
  • Diclaration - Ctrl+B
  • Implementaion - Ctrl+Alt+B

  • Super method - Ctrl+U

  • Test - Ctrl+Shift+T

  • Bookmark - F11/Ctrl+F11

  • Show - Shift+F11

  • Structure + Ctrl+F12

  • Type + Ctrl+H

  • Next Error - F2

  • Override method - Ctrl+O

  • Surround - Ctrl+Alt+T

  • Live templ + Ctrl+J

  • Reformat + Alt + L

  • Auto indent + Ctrl+Alt+I

  • DoubleShift - Search everywhere

  • Refactor this - Alt+R
