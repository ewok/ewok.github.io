Если неправильно отображается кодировка на сайте(для админов)
Столкнулся с проблемой вывода текста из базы данных mysql. Исходные данные: База mysql в utf-8 Сайт в windows-1251 Сайт выводится нормально, а данные из базы данных в виде: РљСЂР°С‚РєР°С. Решение: В .htaccess добавляем строчку “AddDefaultCharset windows-1251”. Апачу дается команда выводить сайт в кодировке windows-1251. Если у вас сайт на php. То после подключения к базе данных нужно сделать mysql_set_charset( ‘cp1251’ );. Или дать каким либо способом запрос базе данных SET NAMES cp1251.
